# GrabGo Packages

This directory contains all the packages for the GrabGo monorepo.

## Packages

- **[grab_go_customer](./grab_go_customer/)** - Customer-facing mobile application
- **[grab_go_rider](./grab_go_rider/)** - Delivery driver mobile application
- **[grab_go_restaurant](./grab_go_restaurant/)** - Restaurant management web panel
- **[grab_go_admin](./grab_go_admin/)** - Admin web dashboard
- **[grab_go_shared](./grab_go_shared/)** - Shared components, assets, and utilities

Each package has its own README with detailed information about features, structure, and usage.
